<li>
2面ダイス
<ul>
<li>
D6 の出目が<code class="language-plaintext highlighter-rouge">{1, 2, 3}</code>のとき<code class="language-plaintext highlighter-rouge">1</code>、<code class="language-plaintext highlighter-rouge">{4, 5, 6}</code>のとき<code class="language-plaintext highlighter-rouge">2</code>となるダイスです。

3面ダイス
<ul>

</li>
</ul>
</li>
<li>
4面ダイス
<ul>
<li>

</li>
</ul>
</li>
<li>
5面ダイス
<ul>
<li>
D10 の出目 ÷ 2 が出目となるダイスです。（割り切れない場合は繰り上げます。）
</li>
<li>
D10 の出目が<code class="language-plaintext highlighter-rouge">0</code>のとき<code class="language-plaintext highlighter-rouge">5</code>となります。
</li>
</ul>
</li>
</ul>

<ul>
<li>
20面ダイス
<ul>
<li>
D10 と D6 を使用します。
</li>
<li>
D6 が<code class="language-plaintext highlighter-rouge">{1, 2, 3}</code>のとき D10 が出目となります。D10 が<code class="language-plaintext highlighter-rouge">0</code>のときは、<code class="language-plaintext highlighter-rouge">10</code>が出目となります。
</li>
<li>
D6 が<code class="language-plaintext highlighter-rouge">{4, 5, 6}</code>のとき D10 + 10 が出目となります。D10 が<code class="language-plaintext highlighter-rouge">0</code>のときは、<code class="language-plaintext highlighter-rouge">20</code>が出目となります。
</li>
</ul>
</li>
<li>
100面ダイス
<ul>
<li>
D10 と D10 を使用します。
</li>
<li>
ひとつ目の D10 が1桁目、ふたつ目の D10 が2桁目を表します。
</li>
<li>
ふたつとも出目が<code class="language-plaintext highlighter-rouge">0</code>のときは、<code class="language-plaintext highlighter-rouge">100</code>が出目となります。
</li>
</ul>
</li>
</ul>
